This is a fully responsive E-commerce web application built using Next.js and TypeScript. Users can browse and explore products across multiple categories including clothing, jewelry, and electronics.

Each product has a dedicated dynamic details page where users can view full product information such as image, description, price, and category. The platform includes advanced filtering by category and price range, along with pagination for better product navigation and performance optimization.

The project follows a clean, scalable, and component-based architecture with strong type safety using TypeScript.
